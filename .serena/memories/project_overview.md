# Phonica プロジェクト概要

## プロジェクトの目的

Phonica は、フィールドレコーディング素材を効率的に管理するための Web アプリケーションです。

### 主要機能

- **音声素材管理**: 録音ファイルのアップロード、メタデータ管理、検索
- **位置情報管理**: GPS座標、地図表示による録音場所の記録
- **機材管理**: 使用した録音機材の記録とマスターデータ管理
- **タグ付け**: 自由なタグ付けによる柔軟な分類
- **プロジェクト管理**: 複数の素材をプロジェクト単位でグループ化
- **メタデータ自動抽出**: 音声ファイルから技術情報を自動取得

### ターゲットユーザー

- フィールドレコーディングアーティスト
- サウンドデザイナー
- 環境音収集家

## プロジェクトの特徴

- モダンな Next.js 15 + React 19 を使用
- TypeScript による型安全な開発
- テスト駆動開発（TDD）の実践
- 高いテストカバレッジ（80%以上必須）
- Chrome専用E2Eテストによる信頼性確保
